Suppression des données

Pour supprimer les données liées à votre Page/compte dans l’application Data Ligue 1, suivez l’une des options ci-dessous.

Option A — Révocation via Meta

  1. Ouvrez Facebook avec le compte administrateur de la Page.
  2. Allez dans Paramètres > Intégrations d’entreprise (ou Paramètres > Sécurité > Applications et sites).
  3. Trouvez Data Ligue 1 et cliquez sur Supprimer / Révoquer l’accès.
  4. Cette action invalide immédiatement le jeton d’accès et supprime l’accès de l’application.

Option B — Demande par e‑mail

Envoyez un message à hello@dataligue1.fr avec :

  • Le nom de votre Page Facebook et/ou l’ID de compte Instagram professionnel.
  • La mention « Suppression des données — Data Ligue 1 ».

Nous révoquerons les jetons, purgerons les journaux techniques et confirmerons la suppression par retour d’e‑mail sous 7 jours ouvrés.

Si vous rencontrez un problème, contactez-nous à hello@dataligue1.fr.
